# AI Dropout Prediction System

## 📌 Overview
This is a frontend web application built using React and Vite. The project focuses on creating a fast, responsive, and user-friendly interface.

## 🚀 Features
- Responsive UI design
- Fast performance using Vite
- Modular component-based architecture
- Easy to maintain and scalable

## 🛠️ Technologies Used
- React.js
- Vite
- JavaScript
- HTML & CSS
- Node.js (for package management)

## 📂 Project Structure
- src/ → Main application code
- public/ → Static files
- components/ → Reusable UI components

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/IshwarirMhaske/AI-Dropout-prediction-system.git

2. Navigate to the project folder:
   cd AI-Dropout-prediction-system

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev
